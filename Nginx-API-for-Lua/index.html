еҹәжң¬з”Ёжі•

  • ctx е°ҶLuaиЎЁжҢҮе®ҡдёәеӯҗиҜ·жұӮзҡ„ngx.ctxиЎЁгҖӮ е®ғеҸҜд»ҘжҳҜеҪ“еүҚиҜ·жұӮзҡ„ngx.ctxиЎЁпјҢеҸҜд»ҘдҪҝеҪ“еүҚиҜ·жұӮе’ҢеӯҗиҜ·жұӮе…ұдә«е®Ңе…ЁзӣёеҗҢзҡ„дёҠдёӢж–ҮиЎЁгҖӮ
  • vars еҸ–еҫ—дёҖдёӘLua tableпјҢиҜҘtableдҝқеӯҳе°ҶеӯҗиҜ·жұӮдёӯжҢҮе®ҡзҡ„NginxеҸҳйҮҸи®ҫзҪ®дёәиҜҘйҖүйЎ№зҡ„еҖјзҡ„еҖјгҖӮ
  • copy_all_vars жҢҮе®ҡжҳҜеҗҰе°ҶеҪ“еүҚиҜ·жұӮзҡ„жүҖжңүNginxеҸҳйҮҸеҖјеӨҚеҲ¶еҲ°еӯҗиҜ·жұӮдёӯгҖӮеӯҗиҜ·жұӮдёӯеҜ№NginxеҸҳйҮҸзҡ„дҝ®ж”№дёҚдјҡеҪұе“ҚеҪ“еүҚиҜ·жұӮгҖӮ
  • share_all_vars жҢҮе®ҡжҳҜеҗҰдёҺеҪ“еүҚиҜ·жұӮе…ұдә«еӯҗиҜ·жұӮзҡ„жүҖжңүNginxеҸҳйҮҸгҖӮеӯҗиҜ·жұӮдёӯеҜ№NginxеҸҳйҮҸзҡ„дҝ®ж”№е°ҶеҪұе“ҚеҪ“еүҚиҜ·жұӮгҖӮ еҗҜз”ЁжӯӨйҖүйЎ№еҸҜиғҪдјҡз”ұдәҺдёҚиүҜзҡ„еүҜдҪңз”ЁиҖҢеҜјиҮҙйҡҫд»Ҙи°ғиҜ•зҡ„й—®йўҳпјҢе№¶дё”иў«и®ӨдёәжҳҜжңүе®ізҡ„гҖӮд»…еңЁе®Ңе…ЁзҹҘйҒ“иҮӘе·ұеңЁеҒҡд»Җд№Ҳж—¶жүҚеҗҜз”ЁжӯӨйҖүйЎ№гҖӮ
  • always_forward_body еҰӮжһңи®ҫзҪ®дёәtrueпјҢеҲҷеҰӮжһңжңӘжҢҮе®ҡbodyйҖүйЎ№пјҢеҲҷеҪ“еүҚпјҲзҲ¶пјүиҜ·жұӮзҡ„иҜ·жұӮдё»дҪ“е°Ҷе§Ӣз»ҲиҪ¬еҸ‘еҲ°жӯЈеңЁеҲӣе»әзҡ„еӯҗиҜ·жұӮгҖӮ з”ұngx.req.read_bodyпјҲпјүжҲ–lua_need_request_bodyиҜ»еҸ–зҡ„иҜ·жұӮжӯЈж–Үе°ҶзӣҙжҺҘиҪ¬еҸ‘еҲ°еӯҗиҜ·жұӮпјҢиҖҢж— йңҖеңЁеҲӣе»әеӯҗиҜ·жұӮж—¶еӨҚеҲ¶ж•ҙдёӘиҜ·жұӮжӯЈж–Үж•°жҚ®пјҲж— и®әиҜ·жұӮжӯЈж–Үж•°жҚ®жҳҜзј“еӯҳеңЁеҶ…еӯҳзј“еҶІеҢәиҝҳжҳҜдёҙж—¶ж–Үд»¶дёӯпјү гҖӮ й»ҳи®Өжғ…еҶөдёӢпјҢжӯӨйҖүйЎ№дёәfalseпјҢе№¶дё”еҪ“жңӘжҢҮе®ҡbodyйҖүйЎ№ж—¶пјҢд»…еҪ“еӯҗиҜ·жұӮйҮҮз”ЁPUTжҲ–POSTиҜ·жұӮж–№жі•ж—¶жүҚиҪ¬еҸ‘еҪ“еүҚпјҲзҲ¶пјүиҜ·жұӮзҡ„иҜ·жұӮжӯЈж–ҮгҖӮ

дҪҝз”ЁжӯӨйҖүйЎ№ж—¶еә”ж јеӨ–е°ҸеҝғпјҢеӣ дёәеҸҳйҮҸиҢғеӣҙе…ұдә«еҸҜиғҪдјҡдә§з”ҹж„ҸеӨ–зҡ„еүҜдҪңз”ЁгҖӮ йҖҡеёёжңҖеҘҪдҪҝз”ЁargsпјҢvarsжҲ–copy_all_varsйҖүйЎ№гҖӮ
